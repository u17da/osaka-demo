# まなびポケット UIモック

大阪市向けのUIモックデモ環境です。各機能のプロトタイプをご確認いただけます。

## 🎯 概要

まなびポケットの新機能プロトタイプを確認できるデモサイトです。

## ✨ 収録コンテンツ

### 1. ナレッジ共有・校務連携（公開中）
生徒カルテ・校務システムと連携し、AIが先生の業務をサポートする教師向けダッシュボード。

**主な機能:**
- 生徒情報の確認（学校生活状況、学習状況、留意事項）
- 声かけ例の提案
- 保護者連絡文の作成
- スクールカウンセラー相談メモの作成
- 心の健康観察の推移分析

### 2. 単元振り返り（児童用）- 準備中
理科「磁石の実験」の単元振り返りシート。児童が学習を振り返り、AIコメントを受け取る機能のデモ。

### 3. 単元振り返り（先生用）- 準備中
クラス全体の振り返り状況を確認し、生徒へコメントを送信できる教師向けダッシュボード。

## 🚀 デプロイ方法

### Vercelでのデプロイ

1. このリポジトリをForkまたはClone
2. [Vercel](https://vercel.com)にログイン
3. 「New Project」をクリック
4. GitHubリポジトリをインポート
5. 設定はデフォルトのまま「Deploy」をクリック

### ローカルでの実行

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/osaka-demo.git
cd osaka-demo

# 依存関係をインストール
npm install

# ローカルサーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。
