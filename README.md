# 先生向けAIアシスタント プロトタイプ

まなびポケット連携を想定した、先生向けAIアシスタントのUIプロトタイプです。

## 🎯 概要

教員が日常業務で活用できるAIアシスタントのデモンストレーション用プロトタイプです。
生徒情報の確認、授業準備支援、保護者連絡文の作成など、様々な機能をシミュレートしています。

## ✨ 主な機能

### 生徒情報管理
- **ピックアップ機能**: 注意が必要な生徒を自動的にハイライト
- **生徒カルテ連携**: まなびポケットの生徒カルテへのダイレクトリンク
- **カテゴリ別情報表示**:
  - 🏫 学校生活状況（出欠、保健室利用、心の健康観察）
  - 📚 学習状況（成績、学習アプリ取り組み状況）
  - 🔔 留意事項（家庭環境、いじめ経験等）

### AIアシスタント機能
- 💬 声かけ例の提案
- 📝 保護者への連絡文作成
- 📚 個別支援計画の作成
- 💚 心の健康観察の推移分析
- 🏥 スクールカウンセラー相談メモの作成

### 授業準備支援
- 時間割連携
- 指導案作成支援
- ナレッジ共有（SharePoint連携）

## 🚀 デプロイ方法

### Vercelでのデプロイ

1. このリポジトリをForkまたはClone
2. [Vercel](https://vercel.com)にログイン
3. 「New Project」をクリック
4. GitHubリポジトリをインポート
5. 設定はデフォルトのまま「Deploy」をクリック

### ローカルでの実行

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/sensei-ai-prototype.git
cd sensei-ai-prototype

# 依存関係をインストール
npm install

# ローカルサーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

## 🎨 デモシナリオ

### 生徒情報の確認フロー

1. 左サイドバーの「⭐ ピックアップ」から「上田亜季さんの最近の様子」をクリック
2. 注意が必要な状況のアラートと、カテゴリ選択ボタンが表示される
3. 「学校生活状況」「学習状況」「留意事項」から詳細を確認
4. 「生徒カルテを開く」ボタンでまなびポケットへの遷移をシミュレート

### 利用可能なデモ生徒

- **上田 亜季さん**（6年1組27番）
  - 遅刻・早退が続いている
  - 心の健康観察で「学校が楽しくない」とコメント
  - 算数が苦手、国語は得意
  - いじめの経験あり

- 実際のデータ連携は行われません
